# My Personal Website

## What This Project Is

This is my personal portfolio website. It shows my work, has an "About Me"
page, and a contact form. It is a simple static website built with HTML,
CSS, and a little bit of JavaScript.

## Important Files

- index.html -- The home page
- about.html -- The "About Me" page
- contact.html -- The contact form page
- css/style.css -- All the styling (colors, fonts, layout)
- js/main.js -- Small scripts for the contact form and navigation
- images/ -- All photos and graphics used on the site

## What I Want Help With

I am not a professional developer, so please:
- Keep the code simple and well-commented
- Explain any changes you make in plain language
- If there are multiple ways to do something, pick the simplest one
- Tell me if something I am asking for is a bad idea and suggest a better way

## Always Do

- Test that pages look right after making changes
- Keep the website fast -- do not add heavy libraries or frameworks
- Make sure the website works on phones and tablets (responsive design)
- Add comments explaining what code does

## Never Do

- Never delete my images or content without asking
- Never add tracking scripts or analytics
- Never change the overall look and feel without showing me first
- Never use complicated frameworks -- keep it as plain HTML/CSS/JS

## Style Preferences

- Colors: I use a blue and white color scheme. The exact colors are in
  css/style.css at the top (look for the CSS variables).
- Fonts: I use "Inter" for body text and "Playfair Display" for headings.
- Layout: Clean and minimal. Lots of white space. No clutter.

## How to Preview

Open index.html in a web browser. That is it -- no build step needed,
no server to start. Just open the file.
